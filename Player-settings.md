# Player Settings

Player settings should support both comfort and situational play. Because the game includes long travel, combat, building, screenshots, social spaces, and large-scale events, players should be able to save multiple named profiles and switch between them quickly.

## Setting Profiles

A profile stores the same kinds of options players expect from a modern PC game, but grouped for different use cases. A combat profile might reduce visual clutter, raise important audio cues, and prioritize performance. A cinematic profile might increase graphics quality, widen ambience, and hide optional UI elements for screenshots. A default profile should remain comfortable for general exploration and settlement life. The heads-up display and interface these profiles adjust are described in [UI and HUD](UI-and-HUD.md).

Profiles should not create competitive advantages beyond letting players choose readable, accessible settings. Any setting that affects fairness, targeting, visibility of hidden enemies, or server-authoritative gameplay should be constrained by the normal rules of the game.

## Audio Layers

Audio settings should be divided into clear layers so players can tune the mix to their needs. Useful layers include master volume, music, ambience, weather, creature cues, combat impacts, voice chat, interface sounds, crafting, settlement ambience, and warning signals. Important danger cues should remain understandable even at lower volumes or through accessibility alternatives. The wider sound design these layers expose, from the adaptive score to combat and creature audio, is set out in [Audio Direction](Audio-direction.md).

## Content and Comfort

Because the game's default presentation is deliberately intense, described in full under [Content and Tone](Content-and-tone.md), players decide how much of it reaches their own screen. A gore intensity control scales blood, dismemberment, and corpse effects from the full presentation down to a restrained minimum, and an optional nudity toggle stays off by default and affects only the local player's view. These remain presentation choices rather than gameplay ones: like every other profile setting they cannot touch fairness, the readability of danger, or anything the server treats as authoritative, so dialling gore down never conceals a threat that another player can still see. These comfort options are part of the broader commitment set out in [Accessibility](Accessibility.md).
